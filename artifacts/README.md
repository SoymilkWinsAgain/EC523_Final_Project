# Artifacts

This directory is for local model indexes and derived artifacts. Git ignores generated contents in this directory.

Typical contents:

- `gallery_index/gallery.faiss`
- `gallery_index/metadata.json`
- `system_test_top500_v2/` for image-only FAISS system tests
- `devise_transform_top500/` for DeVISE text-space indexes and reports
- `joint_clip_top500/` for CLIP/SigLIP/OpenCLIP smoke tests, frozen baselines, comparison metrics, and slide figures
